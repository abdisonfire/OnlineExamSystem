To run this project
Go to ./OnlineExaminationSystem
open settings.py and config database

Execute these commands
```bash
pipenv install
pipenv shell
python manage.py migrate
python manage.py runserver
```
Then go to ./OnlineExaminationSystem-client/oesclient
execute these commands
```bash
npm install
npm start
```
